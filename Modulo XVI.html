CSS Basico

Que es CSS

CSS son las siglas de Cascading Style Sheets o en español, hojas de estilo en cascada, se usa para especificar el aspecto de las paginas web, se define en base a un estándar publicado por una organización llamada W3C, que también se encarga de estandarizar el propio lenguaje HTML.

Agregar colores a Texto

Para poder cambiar el color de un texto dentro de CSS primero hay que escribir a que etiqueta, clase o id se le va a cambiar el color después de esto hay que escribir “Color: ;” después de los dos puntos hay que especificar el color que se le quiere aplicar al texto, esto se puede hacer de varias maneras, ya sea escribiendo el nombre del color en inglés, o definido el color en sistema sexagesimal, en incluso en el sistema RGB

Cambiar tamaño de Imagenes

Para poder cambiar el tamaño de una imagen o cualquier otro elemento, primero hay que escribir un punto después la etiqueta, clase o id que se le haya dado a la imagen anteriormente, una imagen o elemento se le puede modificar tanto el ancho como el alto, asi que para modificar el ancho se utiliza “width:. ;” y para modificar el alto se utilizar “height: ;” después de escribir uno de estos dos se le espática el tamaño que se le quiere dar ya se en número de pixeles o en un porcentaje

Agregar y Modificar un borde con CSS

Para poder agregar un borde ya sea a una imagen texto etc. y poder modificarlo, primero hay que escribir un punto y poder la etiqueta, clase o id que se le haya dado anteriormente, después hay que agregar “border: ;” después de los dos puntos se agrega que tipo de borde se quiere agregar, hay varios tipos, “solid” que coloca un borde normal , también está “dotted” que agrega un borde punteado, etc. Después se puede especificar que color se le quiere dar al borde esto se puede hacer simplemente agregando el nombre del color que se desea dar al borde, también se le puede cambiar el grosor al borde, esto se hace agregando la cantidad de pixles que se le quiere dar de grosor, también está “border-radius” que modifica las esquinas de los bordes, redondeándolas, la cantidad de redondeo se define en pixeles

Normalize

Que es Normalize

Normalize es un archivo CSS que se usa para poder mantener estilos similares en los navegadores, esto porque cada página agrega un estilo propio por defecto, con Normalize no importa con que navegador se habrá la página web se verá igual, lo que hace Normalize es corregir errores e inconsistencias que generan los navegadores, también normaliza estilos para distintos elementos de HTML además mejora algunas otras cosas de manera sutil

Sitio Web Responsivo

Los sitios web responsivos están hechos para adaptarse a distintos tipos de equipos como tablets y Smartphone etc. Las páginas web responsivas adaptan su contenido al tamaño de pantalla del dispositivo, ajustando automáticamente todos los elementos para desplazarse en vertical, esto incluye tanto texto, como imágenes
