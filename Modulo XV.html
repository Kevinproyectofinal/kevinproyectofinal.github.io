Estructura de una Pagina Web

Una página web, está compuesta de 2 partes: El encabezamiento y el cuerpo de la página. También existen tres etiquetas fundamentales, estas deben estar incluidas en el archivo HTML de manera obligatoria, las etiquetas son HTML, HEAD, BODY, HTML le dice al navegador que el documento es un documento en HTML, esta es la que se pone de primero dentro del archivo y es la última etiqueta en cerrarse, HEAD es la parte donde se detalla el encabezado de la página, esta se abre después de la de HTML, BODY es el cuerpo de la página, es donde va el contenido esta se abre después de HEAD.

Formularios en HTML

Para poder hacer un formulario hay que agregar la etiqueta FORM dentro de esta etiqueta se pueden agregar cosas como Inputs que le permitirán al usuario ingresar datos, dentro del formulario podemos agregar etiquetas como TEXTBOX, PASSWORD, CHECKBOXES y otras muchas más etiquetas

Esta etiqueta nos permite hacer que el usuario pueda subir un archivo desde su computadora hacia la página.
